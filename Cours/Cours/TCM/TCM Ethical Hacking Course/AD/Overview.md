# Overview
95% of fortune 1000 use AD
AD is a directory service  written by Microsoft to manage Windows domain network
Stores informations about windows objects such as computers, users  printers, it's a phne book for windows
Autenticates using Kerberos tickets, non Windows devices such as linux, firewall etc.. can antenticate via RADIUS or LDAP
It's an identity manager with wich you can modify password length for exemple
Can be exploited without attacking patchable exploits, instead we abuse features, components, trusts and more
