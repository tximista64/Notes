# Mobile Security
La **Mobile Security** vise à protéger les appareils mobiles (smartphones, tablettes), les données qu’ils contiennent, ainsi que les réseaux auxquels ils se connectent. Ces appareils sont de véritables **coffres-forts numériques** contenant des données personnelles et professionnelles sensibles.
---
## Analogie : le coffre au trésor
Ton **smartphone** = un **coffre** que tu portes partout :
- Lettres privées, photos de famille, documents bancaires, contrats pro, etc.
- Un trésor convoité par les cybercriminels
---
## Objectif
- **Protéger les données stockées**
- **Sécuriser les connexions**
- **Empêcher les accès non autorisés**
- **Contrôler les applications installées**
---
## Axes de sécurité
| Domaine               | Mesures principales |
|-----------------------|---------------------|
| **Device Security**   | Code PIN, biométrie, verrouillage auto, remote wipe |
| **Data Security**     | Chiffrement, sauvegardes, DLP |
| **Network Security**  | VPN, HTTPS, blocage Wi-Fi non sécurisé |
| **Application Security** | App vetting, permissions, dev sécurisé |
---
## Détail par couche
### 1. Device Security
- **Mot de passe / Biométrie**
- **Verrouillage automatique**
- **Remote Wipe** : effacement à distance en cas de perte/vol
### 2. Data Security
- **Chiffrement des données**
- **Sauvegardes sécurisées**
- **DLP (Data Loss Prevention)** : éviter les fuites accidentelles
### 3. Network Security
- **VPN** : tunnel chiffré
- **Protocoles sécurisés** : HTTPS, TLS
- **Éviter les Wi-Fi publics non sécurisés**
### 4. Application Security
- **Vérification des apps avant installation**
- **Gestion des permissions**
- **Développement sécurisé côté éditeurs**
---
## Responsabilités
| Acteur                      | Rôle |
|-----------------------------|------|
| **IT Departments**          | Déploiement d’outils (MDM, VPN, chiffrement) |
| **CISO**                    | Stratégie globale, évaluation des risques |
| **Security Teams**          | Tests, audits, pentests mobile |
| **IT Security Managers**    | Supervision quotidienne, adaptation des défenses |
---
## Conclusion
Les appareils mobiles sont les **extensions numériques** de notre vie privée et professionnelle. Leur sécurité est devenue **indispensable** pour :
- Protéger les données sensibles
- Éviter les compromissions via réseaux/applications
- Garantir la conformité et la résilience organisationnelle
La Mobile Security est une brique **essentielle** de toute stratégie InfoSec moderne.
