# Introduction to Network Log Analysis


**tags**:  #log #réseau

The components that make up the concept of the Internet and its 
infrastructure are network devices. It would not be possible to talk 
about the internet without the network devices which we keep using with 
different roles and categories day by day. In the past, a network 
communication reached its destination by passing through devices such as
 hubs, switches, and routers, but today, many devices with different 
tasks (such as Firewall, IPS/IDS, Proxy, and WAF) come into play with 
different tasks and capabilities.

Within the scope of this training, we will cover the analysis of the 
logs of important network devices and services that carry the traces 
within the network communications.

At the end of the training, you will learn how to analyze logs of 
important network equipment that a SOC Analyst needs to know. The logs 
shared during the training are shared as an example. Each device, 
product, application logs are unique and may differ according to the 
examples within the scope of training. The important thing here is to 
understand the essence of the subject.
