# Dashboards

**tags**:  #siem #soc #splunk 

A dashboard is nothing more than a dashboard. Here you can find a lot of 
information you need.
You can make one for analysts, and one for your customers, with 
different types of information. If you use always the same requests and 
always need the same information, then making a Dashboard is a good 
idea.
